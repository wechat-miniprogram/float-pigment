# HTML Test Framework Plan A: Codegen + Migration

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Migrate float-pigment-layout's 970 `assert_xml!` tests to HTML files + build.rs codegen, so `cargo test` runs generated tests from HTML (behavior unchanged).

**Architecture:** HTML files in `tests/cases/<topic>/<case>.html` carry `data-expect-*` attributes. A `build.rs` in `float-pigment-forest` scans them and generates `#[test]` functions (each calls `TestCtx::from_str` → `layout` → `assert`). `TestCtx::set_expect_layout_pos` is extended to read `data-expect-*` (alongside legacy `expect_*`). A conversion script bulk-migrates existing `assert_xml!` tests to HTML.

**Tech Stack:** Rust (build.rs codegen), float-pigment-forest `TestCtx`, existing `assert_xml!` internals.

**Spec:** `docs/superpowers/specs/2026-07-17-html-test-framework-design.md`

`float-pigment-forest/build.rs`:
```rust
use std::{fs, path::PathBuf};

fn main() {
    let cases_dir = PathBuf::from("tests/cases");
    let mut tests = String::new();
    tests.push_str("// AUTO-GENERATED by build.rs. Do not edit.\n");
    tests.push_str("use crate::*;\n\n");
    if !cases_dir.exists() {
        // No cases yet — emit empty so include! still works.
        let out = std::env::var("OUT_DIR").unwrap();
        fs::write(format!("{out}/html_tests.rs"), tests).unwrap();
        return;
    }
    let mut entries: Vec<_> = walk(&cases_dir, &cases_dir);
    entries.sort();
    for (rel, abs) in entries {
        let ext = abs.extension().and_then(|e| e.to_str()).unwrap_or("");
        if ext != "html" { continue; }
        let fn_name = rel.to_string_lossy().replace(['/', '.', '-'], "_");
        let abs_str = abs.to_str().unwrap();
        tests.push_str(&format!(
            "#[test]\nfn html__{fn_name}() {{\n  assert_html_case!(\"{abs_str}\");\n}}\n\n"
        ));
    }
    let out = std::env::var("OUT_DIR").unwrap();
    fs::write(format!("{out}/html_tests.rs"), tests).unwrap();
    println!("cargo:rerun-if-changed=tests/cases");
}

fn walk(root: &PathBuf, dir: &PathBuf) -> Vec<(PathBuf, PathBuf)> {
    let mut out = Vec::new();
    if let Ok(rd) = fs::read_dir(dir) {
        for e in rd.flatten() {
            let p = e.path();
            if p.is_dir() {
                out.extend(walk(root, &p));
            } else {
                let rel = p.strip_prefix(root).unwrap().to_path_buf();
                out.push((rel, p));
            }
        }
    }
    out
}
```

### Task 4: Conversion script (assert_xml! → HTML)

**Files:**
- Create: `scripts/convert-assert-xml.mjs`

- [ ] **Step 1: Write the conversion script**

`scripts/convert-assert-xml.mjs` (node, no deps — uses regex):
```javascript
// Converts a Rust test file of `assert_xml!(r#"..."#)` cases into HTML files.
// Usage: node scripts/convert-assert-xml.mjs <input.rs> <output-dir> <topic>
import { readFileSync, mkdirSync, writeFileSync } from 'node:fs';
import { join } from 'node:path';

const [, , inputPath, outDir, topic] = process.argv;
const src = readFileSync(inputPath, 'utf8');

// Match: #[test]\n fn name() { assert_xml!(r#"XML"#) }
const re = /#\[test\][\s\S]*?fn\s+(\w+)\s*\(\s*\)\s*\{[\s\S]*?assert_xml!\(\s*r#"([\s\S]*?)"#\s*(?:,\s*(true|false))?\s*\)/g;
let m;
let count = 0;
while ((m = re.exec(src)) !== null) {
  const name = m[1];
  const xml = m[2].trim();
  // expect_* -> data-expect-*
  const body = xml.replace(/expect_(width|height|left|top)/g, 'data-expect-$1');
  // wrap in a single root: the assert_xml XML usually has a single outer div.
  const html = `<!DOCTYPE html>\n${body}\n`;
  mkdirSync(join(outDir, topic), { recursive: true });
  writeFileSync(join(outDir, topic, `${name}.html`), html);
  count++;
}
console.error(`converted ${count} cases from ${inputPath} -> ${outDir}/${topic}/`);
```
